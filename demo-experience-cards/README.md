# Experience Cards — Design Demo

Standalone React + Vite + Tailwind demo showing **6 experience card layouts** on one page. Uses your blue theme.

## Run

```bash
cd demo-experience-cards
npm install
npm run dev
```

Open the URL shown (e.g. http://localhost:5173).

## What’s on the page

1. **Idea 1: Left accent bar** — Compact card with vertical accent strip
2. **Idea 2: Logo strip** — Full-height logo strip on the left
3. **Idea 3: Timeline** — Vertical timeline with dot + line
4. **Idea 4: Bento grid** — Responsive grid of equal-height cards
5. **Idea 5: Accordion** — Collapsed row, expand for details
6. **Idea 6: Role badge** — Role as pill, minimal card

Logos are placeholders (initials). After you pick a style, we can wire it into your main site and use real logos.
